Admin AvantMoi

Connexion admin via ton compte Google autorisé.

Connexion requise.

Si la connexion echoue (Cloudflare)
  • Dans Workers & Pages → avantmoi → Settings → Variables, definis en Production : ADMIN_GOOGLE_CLIENT_ID (client OAuth Web), ADMIN_GOOGLE_EMAILS (ton email exact), ADMIN_SESSION_SECRET (chaine aleatoire longue).
  • Google Cloud Console → identifiants OAuth : Origines JavaScript autorisees = https://avantmoi.com (et la preview Pages si besoin).
  • Redeploie ou attend la prise en compte des variables, puis recharge cette page.

Stats de consultation (Google Analytics)

Apres connexion Google, charge la liste de tes proprietes GA4 puis choisis-en une. Tu peux aussi saisir l ID a la main.

API Google Cloud

Active Google Analytics Admin API sur le meme projet que ton client OAuth (APIs & Services → Bibliotheque). Le scope analytics.readonly suffit pour lister les proprietes et lire les rapports.
